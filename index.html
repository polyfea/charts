Polyfea Helm Charts

Official Helm Repository for Polyfea Microfrontend Components

Available Charts

polyfea-controller

Deploys the Polyfea microfrontend controller and its Custom Resource Definitions (CRDs). The core component for managing microfrontends in your Kubernetes cluster.

polyfea-md-shell-samples

Sample applications demonstrating the Polyfea Material Design Shell with web components. Perfect for getting started and learning how Polyfea works.

Getting Started

1. Add the Helm Repository

helm repo add polyfea https://polyfea.github.io/charts helm repo update

2. Search for Available Charts

# List all Polyfea charts helm search repo polyfea # Show all available versions helm search repo polyfea --versions

3. View Chart Information

# Show the chart README helm show readme polyfea/polyfea-controller # View default values helm show values polyfea/polyfea-controller

4. Install a Chart

# Install the Polyfea controller helm install polyfea-controller polyfea/polyfea-controller \ --namespace polyfea-system \ --create-namespace # Or install with custom values helm install polyfea-controller polyfea/polyfea-controller \ --namespace polyfea-system \ --create-namespace \ --set replicaCount=2
